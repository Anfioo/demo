快速入门

如何登录教务系统？
1. 点击右上角的"登录"按钮
2. 输入学号和初始密码（默认为身份证后6位）
3. 首次登录需要修改密码
4. 完成后即可进入系统
系统支持哪些浏览器？
推荐使用以下浏览器访问系统：
- Chrome 80及以上版本
- Firefox 75及以上版本
- Edge 80及以上版本
- Safari 13及以上版本
注意：不建议使用IE浏览器
如何查看系统公告？
1. 登录后默认进入首页
2. 在首页可以看到最新公告
3. 点击"查看更多"可以浏览历史公告
4. 重要公告会以红色标记显示

账号管理

如何修改密码？
1. 登录后点击右上角的用户头像
2. 选择"修改密码"选项
3. 输入原密码和新密码
4. 新密码必须包含字母和数字，长度8-20位
5. 点击确认即可修改成功
忘记密码怎么办？
方式一：通过邮箱找回
1. 点击登录页面的"忘记密码"
2. 输入学号和注册邮箱
3. 获取验证码并输入
4. 设置新密码

方式二：教务处重置
1. 携带学生证到教务处
2. 填写密码重置申请表
3. 工作人员核实身份后重置
如何绑定邮箱？
1. 登录后进入"个人信息"页面
2. 点击"绑定邮箱"按钮
3. 输入常用邮箱地址
4. 获取验证码并输入
5. 验证成功后即完成绑定
注意：绑定邮箱后可用于接收重要通知和找回密码

选课指南

选课的基本流程是什么？
1. 登录系统后点击"选课系统"
2. 查看选课通知和选课指南
3. 在规定时间内进行选课
4. 选课分为以下阶段：
- 预选阶段：可选意向课程
- 正选阶段：确认选课
- 补退选阶段：可补选或退选课程
5. 选课结束后可打印选课结果单
如何查看课程信息？
1. 进入"课程查询"页面
2. 可按以下条件筛选：
- 课程名称
- 课程代码
- 任课教师
- 上课时间
- 上课地点
3. 点击课程可查看详细信息：
- 课程大纲
- 教材信息
- 考核方式
- 选课要求

考试相关

如何查看考试安排？
1. 进入"考试安排"页面
2. 可查看以下信息：
- 考试科目
- 考试时间
- 考试地点
- 座位号
3. 可导出或打印考试安排表
注意：考试前请及时查看是否有变动
如何查询成绩？
1. 进入"成绩查询"页面
2. 选择学期或学年
3. 可查看以下内容：
- 课程成绩
- 绩点统计
- 成绩分布
- 历史成绩
4. 可导出成绩单
